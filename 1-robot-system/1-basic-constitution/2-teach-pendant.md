# 1.1.2 티치 펜던트

티치 펜던트는 TP600과 TP630을 지원합니다. 이 조작 설명서에서는 TP600 모델을 기준으로 사용 방법을 설명합니다.

TP600은 Hi6 제어기 전용으로 개발된 모델로 큰 화면의 터치 스크린을 제공합니다.
